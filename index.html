Intro Video Script Generator (for Google Colab)

How it Works:

  1. Select the photos and videos you want in the intro grid.
  2. Enter the total desired duration for the intro video (in seconds).
  3. Click "Generate Script".
  4. The app will show you the detected filenames (Important!) and generate a Python script.
  5. Copy the generated Python script.
  6. Open Google Colab (colab.research.google.com).
  7. In Colab, use the 'Files' tab on the left to upload the exact same photos/videos with the exact same filenames (case-sensitive!). Optionally, also upload `narration.mp3` and `background_music.mp3` if you want audio.
  8. Paste the copied script into a Colab code cell and run it.
  9. Wait for Colab to process (it might take minutes).
  10. Download the final `final_intro.mp4` from Colab's file browser.
